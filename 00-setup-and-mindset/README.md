# 00 - Setup & Mindset

Purpose: set the operating mindset before starting any VAPT workflow.

Use this module before touching a target:

- Confirm scope and authorization.
- Define the target, time limit, and rules of engagement.
- Prepare a notes file for commands, findings, and evidence.
- Work in loops: enumerate, identify, exploit, escalate.
- Do not run tools blindly. Every command should answer a question.

Output from this module:

- Clear scope
- Basic target plan
- Ready-to-use attack notes
