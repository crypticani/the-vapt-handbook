# 07 — CTF Writeups: Labs

> Your CTF practice starts here.

## 🧪 Lab 1: Complete a TryHackMe Room

1. Sign up at https://tryhackme.com
2. Complete: **OWASP Top 10** room (free)
3. Write a full writeup using the template from `notes.md`
4. Save as `writeups/tryhackme-owasp-top10.md`

## 🧪 Lab 2: Solve 5 PicoCTF Challenges

1. Go to https://picoctf.org
2. Solve 5 challenges (mix of categories)
3. Write a writeup for each
4. Focus on explaining WHY your solution works

## 🧪 Lab 3: Compete in a Live CTF

1. Check https://ctftime.org for upcoming CTFs
2. Register (solo or find a team)
3. Solve at least 1 challenge
4. Write a detailed writeup

## 🧪 Lab 4: HackTheBox Machine

1. Sign up at https://hackthebox.com
2. Complete an Easy-rated machine
3. Write a full machine writeup covering:
   - Enumeration
   - Initial foothold
   - Privilege escalation
   - Lessons learned
