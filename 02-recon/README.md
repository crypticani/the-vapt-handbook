# 02 - Recon

This module turns targets into testable facts.

## Outcomes

- discover hosts, endpoints, and hidden inventory
- separate public surface from internal or preview surface
- identify technologies, frameworks, API styles, and likely trust boundaries
- produce a prioritized attack surface instead of raw scanner output
