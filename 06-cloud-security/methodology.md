# 06 — Cloud Security: Methodology

> See [labs.md](labs.md) for the integrated cloud pentest framework methodology.

## 🔴 Quick Reference

```
CLOUD PENTEST FLOW:
1. Enumerate identity → What account/role am I?
2. Enumerate permissions → What can I do?
3. Discover resources → What exists?
4. Find misconfigurations → What's exposed/weak?
5. Exploit → Escalate, exfiltrate, persist
6. Report → Findings + IaC remediation
```
