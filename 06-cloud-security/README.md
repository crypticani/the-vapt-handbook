# 06 - Cloud Security

This module focuses on cloud-linked attack paths that matter during appsec and pentests.

## Outcomes

- enumerate IAM, storage, workloads, and secrets exposure
- turn SSRF or leaked credentials into cloud impact
- audit Kubernetes and workload identity trust
- connect app flaws to AWS or cloud-native follow-on risk
